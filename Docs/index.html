Privacy Policy

Outlook Hover Card Blocker Chrome Extension

Last updated: August 4, 2025

TL;DR: This extension doesn't collect, store, transmit, or access any of your personal data. It simply hides hover cards on Outlook.com using CSS and local browser functionality.

What This Extension Does

The Outlook Hover Card Blocker extension enhances your Outlook.com experience by:

  • Adding CSS styles to hide contact hover cards
  • Removing title attributes from contact elements to prevent tooltips
  • Monitoring the page for new elements to apply the same hiding functionality
  • ...thats all it does, no external connections, no nothing

Data Collection and Privacy

We collect absolutely no data. This extension:

  • ✅ Does NOT collect any personal information
  • ✅ Does NOT track your browsing activity
  • ✅ Does NOT access your emails, contacts, or calendar
  • ✅ Does NOT send any data to external servers
  • ✅ Does NOT use cookies or local storage
  • ✅ Does NOT require user accounts or authentication

How It Works

The extension operates entirely within your browser and only affects the visual presentation of Outlook.com pages. It:

  • Injects CSS styles to hide specific hover card elements
  • Modifies HTML attributes locally in your browser
  • Uses browser APIs only to detect when new page elements are added

All processing happens locally on your device. No information ever leaves your browser.

Permissions Explained

The extension requests these minimal permissions:

  • "activeTab": Allows the extension to work on the current Outlook.com tab you're viewing
  • Host permissions for Outlook domains: Allows the extension to run only on Outlook.com, outlook.office.com, and outlook.office365.com

These permissions are the minimum required for the extension to function and do not grant access to your personal data.

Third-Party Services

This extension does not integrate with, connect to, or communicate with any third-party services, analytics platforms, or external servers.

Data Security

Since no data is collected, stored, or transmitted, there are no data security concerns. The extension operates entirely within Chrome's secure extension environment.

Children's Privacy

This extension does not collect any information from anyone, including children under 13.
